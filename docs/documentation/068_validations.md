# Validations

Les validations sur les attributs de modèles sont basées sur les validations d'Active Record, et sont retranscries à travers l'interopérabilité.

::: warning Important 
Consulter la documentation d'[Active Record Validations](https://guides.rubyonrails.org/active_record_validations.html) pour en savoir plus
:::


## Validations sur les pièces-jointes

Les validations sur les pièces-jointes sont disponible à travers la Gem `active_record_validations`.


::: warning Important 
Consulter la documentation de la [Gem](https://github.com/igorkasyanchuk/active_storage_validations)
:::
