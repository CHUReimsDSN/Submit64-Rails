# Installation

``` ruby
# Gemfile
gem 'submit64', git: 'https://github.com/CHUReimsDSN/Submit64-Rails.git'
```

```sh
bundle install
```


## Cibler une branche spécifique

``` ruby
# Gemfile
gem 'submit64', git: 'https://github.com/CHUReimsDSN/Submit64-Rails.git', branch: 'nom_de_la_branche'
```

## Dépendances

```ruby
# Runtime depedencies
"activerecord", ">= 6.0"
```
